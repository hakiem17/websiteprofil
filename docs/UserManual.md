# Panduan Pengguna Admin Website Pemerintah Daerah

Dokumen ini berisi panduan untuk operator/admin dalam mengelola konten website melalui Dashboard Admin.

## 1. Akses Admin
1. Buka browser dan kunjungi `https://domain-anda.go.id/login`.
2. Masukkan **Email** dan **Password** admin yang telah didaftarkan.
3. Klik tombol **Masuk Dashboard**.

## 2. Dashboard Overview
Setelah login, Anda akan diarahkan ke halaman Dashboard utama yang menampilkan:
- Statistik ringkas (Total Dokumen, Berita, Layanan).
- Tombol akses cepat (Aksi Cepat) untuk menambah konten.
- Aktivitas terbaru.

## 3. Mengelola Berita & Artikel
Menu: **Berita & Artikel**

### Menambah Berita Baru
1. Klik tombol **+ Tulis Berita** di pojok kanan atas.
2. Isi formulir:
   - **Judul Berita**: Judul yang menarik dan jelas.
   - **Slug**: Terisi otomatis, bisa diedit jika perlu (hindari spasi).
   - **Konten**: Tulis isi berita.
   - **Link Thumbnail**: URL gambar untuk sampul berita.
   - **Status**: Pilih **Terbit** untuk langsung menayangkan, atau **Draf** untuk menyimpannya saja.
3. Klik **Simpan Berita**.

### Mengedit/Menghapus Berita
- Klik ikon **Pensil (Edit)** pada daftar berita untuk mengubah konten.
- Klik ikon **Sampah (Hapus)** untuk menghapus berita.

## 4. Mengelola Dokumen (Arsip Transparansi)
Menu: **Dokumen**

Digunakan untuk mengunggah dokumen seperti Renstra, Renja, DPA, LHKPN, dll.

### Menambah Dokumen
1. Klik tombol **+ Tambah Dokumen**.
2. Isi formulir:
   - **Judul Dokumen**: Nama dokumen (misal: "Renja Dinas 2024").
   - **Kategori**: Pilih kategori yang sesuai (Renstra, DPA, dsb).
   - **Tahun**: Tahun anggaran dokumen.
   - **File URL**: Link ke file dokumen (PDF).
3. Klik **Simpan Dokumen**.

## 5. Mengelola Layanan Publik
Menu: **Layanan**

### Menambah Layanan
1. Klik **+ Tambah Layanan**.
2. Isi Nama dan Deskripsi.
3. Pilih Tipe:
   - **Online**: Wajib mengisi **Link URL** (alamat web layanan).
   - **Offline**: Wajib mengisi **Lokasi** (alamat kantor/gedung).
4. Klik **Simpan**.

## 6. Mengelola Galeri
Menu: **Galeri**

1. Klik **+ Tambah Item**.
2. Masukkan Judul dan **URL Media** (link foto/video).
3. Pilih Tipe (Foto/Video).
4. Klik **Simpan**.
